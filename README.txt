実行前に必ず目を通して下さい。

前提として、この実行ファイルは、Python3系とpyautoguiがインストールされていることが実行の条件となります。

ー実行方法ー
・Macの場合
AutoClicker.commandをダブルクリックして下さい。
権限が無い的なことが出てきた場合は、”ターミナル”を開いて、「chmod 777 ”AutoClicker.command”」と入力しEnter。

・Windowsの場合
Autoclicker.batをダブルクリックして下さい。
実行がうまくいかない場合は、Autoclicker.batを右クリックして、”管理者として実行”をクリックして下さい。
